Server-Side Rendering

Definitions and benefits of SSR

Server-side rendering is the ability of an application to display webpages on the server instead of rendering it in the browser.
Advantages:
SSR enables faster loading and betters the user experience.
Plays a role in SEO as search engines favor pages with faster loading times.
Assists with loading for users with slower internet connections.
Assist with loading for users with outdated devices.
Disadvantages:
Slower page transitions.
Bigger surface area - more vulnerable.
Responsetime dependant on trafict - The more trafic the slower the response.
Can be timeconsuming to render bigger applications.
