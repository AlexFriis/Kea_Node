Javascript Variables


Variables defined with let cannot be redeclared.
Variables defined with let must be declared before use.
Variables defined with let have block scope.

Variables defined with const cannot be redeclared.
Variables defined with const cannot be reassigned.
Variables defined with const have block scope.

In practice we strive to use const and let instead of var as they are limited to the scope whereas var declares a variable globally regardless of scope.

Hoisting is JavaScript's default behaviour of moving all declarations to the top of the current script or function.
This allows us to declare variables after they've been used.
Which can also be turned around to - A variable can be used before it has been declared.

In JavaScript we can declare variables in 4 different ways:


                        //By using var
                        var variable;
                        //By using let
                        let variable;
                        //By using const
                        const variable;
                        //By using nothing
                        x = 5;
                        
                        let x = 4;
                        let x = "String";
                        //SyntaxError

Callbacks

Callback functions can be used to pass an argument to another function.
Allows a function to call another function.
Callback functions can also run subsequently as other functions finish.
Callback functions has various functionality but really shines when used asynchronous where one function has to wait for another. 


                        //Callback example
                        
                        function displayThis(somethingToDisplay) {
                            document.getElementById("idTag").innerHTML = somethingToDisplay;
                        }
                        
                        function myCalc(num1, num2, myCallback) {
                            let sum = num1 + num2;
                            myCallback(sum);
                        }
                        
                        myCalc(5, 5, displayThis);
                        
                        //In this example displayThis is the name of the function which we pass to myCalc() as an argument.

Arrays

Arrays are special variables which can hold more than one value.
Generally used when we have a list of items which we can store under a single variable instead of listing them seperately. 


                        //Array Example
                        
                        const juicearray = ["Apple", "Orange", "Multifruit"]
                        
                        //Array has a multitude of functionality - A few examples
                        
                        //.pop() removes the last element of the array which would be "Multifruit"
                        juicearray.pop();
                        
                        //.push() adds a new element to the end of the array
                        juicearray.push("Melon");
                        //Which would add melon to the juicearray.
                        
                        //Array delete()
                        delete juicearray[0];
                        //Removes the value at index 0 - However using delete() leaves undefined holes in the array which makes other methods prefered such as .pop().

Loops

Loops can be used to execute a block of code numerous times which has many purposes.
An example of this could be to iterate through an array and return the specific value.
As with other code, there are various ways to write it, some approaches are prefered. 


                    //Loop example
                    
                    const cars = ["Fiat", "Opel", "Ford"];
                    
                    for (let i = 0; i < cars.length; i++) {
                        console.log(cars[i]);
                    }
                    
                    //To make this safer, readable and remove potential side-effects we can take another approach.
                    
                    //Another approach
                    const cars = ["Fiat", "Opel", "Ford"];
                    
                    cars.forEach(element => console.log(element));

Fetch

When building web applications, medias etc. we can use Fetch to make HTTP requests serverside and clientside.
This project was made in node with express. 


                    //A simple fetch example
                    
                    fetch('URL')
                    .then(res => res.text())
                    .then(text => console.log(text));
                    
                    //In this example we fetch an URL and add multiple promises which are to be executed once we receive a response.
                    //In this case we await the response on the URL and convert it to text format.
                    //Hereafter we print the text with console.log().
